News Dashboard

This is a simple News Dashboard application built using React. It fetches real-time news data from a public API and allows users to explore articles based on search and category.

Features

- Fetch and display latest news articles
- Search news by keyword
- Filter news by category (Technology, Business, Sports, Health)
- View article details via external link
- Clean and responsive UI using Material UI

Tech Stack

- React (Hooks)
- Vite
- Material UI
- JavaScript 
- GNews API
